# LP Advocacia App

Site institucional premium para advocacia autônoma, desenvolvido em React com foco em apresentação profissional, navegação clara e experiência responsiva.

O projeto foi construído para transmitir autoridade, confiança e sofisticação em um contexto jurídico, preservando uma linguagem visual discreta e institucional. A interface reúne uma página principal completa e páginas específicas para áreas de atuação, com conteúdo direcionado para Direito Cível, Direito Criminal e Direito Trabalhista.

## Visão geral

- Página inicial com hero institucional, seção sobre o advogado, áreas de atuação, processo de atendimento, depoimentos, FAQ, contato e footer.
- Páginas internas para serviços jurídicos em Direito Cível, Criminal e Trabalhista.
- Layout responsivo para desktop, tablet e mobile.
- Identidade visual baseada em tons sóbrios, dourado discreto e tipografia elegante.
- Botões de contato e WhatsApp com microinterações suaves.
- Estrutura modular em componentes React.

## Principais páginas

- `/` - Página principal institucional.
- `/advogadocivel` - Página de atuação em Direito Cível.
- `/advogadocriminalista` - Página de atuação em Direito Criminal.
- `/advogadotrabalhista` - Página de atuação em Direito Trabalhista.

## Tecnologias

- React
- Vite
- React Router
- React Icons
- Tailwind CSS
- CSS modular por seções

## Objetivo visual

O design busca evitar aparência genérica ou excessivamente promocional, priorizando uma experiência humana, elegante e confiável. A estrutura foi pensada para facilitar a leitura, destacar informações relevantes e conduzir o visitante naturalmente para o contato profissional.
